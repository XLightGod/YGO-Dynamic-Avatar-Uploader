# YGO-Dynamic-Avatar-Uploader

Dynamic Avatar Uploader for 一起来决斗

## Requirements

- Windows
- Wechat for Windows
- Python 3.10+
- mitmproxy

## Build

1. Generate the mitmproxy certificate and place it in the `certs` folder.
2. Run `build.bat`.
